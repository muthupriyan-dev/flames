# 💗 FLAMES

## 📌 Overview
A fun web-based FLAMES calculator that predicts relationships based on two names using the classic FLAMES game logic.

## ✨ Features
- Simple name-based FLAMES calculation
- Instant results
- Clean and responsive interface
- Works directly in the browser

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
